Полнофункциональный веб-сайт для кафе "Бар-да-бар"

Современный веб-сайт с возможностью просмотра меню, афиши мероприятий, бронирования столиков и многого другого.

О проекте

Проект представляет собой полнофункциональный веб-сайт для кафе "Бар-да-бар", состоящий из трех основных частей:

  • Frontend (Next.js) - Современный лендинг с SSR
  • Backend (Express.js) - API сервер
  • Admin Panel (Strapi) - Панель управления контентом

Сайт использует следующую цветовую схему:

  • Основной: #7A2E2E (бордовый)
  • Акцент: #F5A623 (золотой)
  • Фон: #FDF8F4 (кремовый)
  • Текст: #333333 (темно-серый)

Frontend (Next.js)

Главная страница

Современный лендинг с секциями:

  • Hero секция с фоновым изображением
  • Секция "О нас"
  • Превью меню
  • Превью мероприятий
  • Отзывы клиентов

Страница меню

Интерактивное меню с возможностями:

  • Фильтрация по категориям
  • Карточки блюд с фото и описанием
  • Поиск и сортировка

Страница мероприятий

Афиша мероприятий с функциями:

  • Календарь событий
  • Фильтрация по типу мероприятия
  • Детальные страницы событий

Страница бронирования

Форма бронирования столиков:

  • Выбор даты и времени
  • Указание количества гостей
  • Валидация данных
  • Подтверждение бронирования

Страница контактов

Контактная информация:

  • Интерактивная карта
  • Контактная форма
  • Режим работы
  • Социальные сети

QR-генератор

Генератор QR-кодов для меню:

  • Настройка внешнего вида
  • Скачивание в разных форматах
  • Инструкции по использованию

Backend (Express.js)

API Endpoints

  • GET /api/menu - получение меню
  • GET /api/events - получение мероприятий
  • GET /api/reviews - получение отзывов
  • POST /api/reviews - добавление отзыва
  • POST /api/reservation - бронирование столика
  • GET /api/reservation/available-times - доступное время
  • POST /api/contact - обработка контактной формы

Особенности

  • REST API для всех функций
  • Интеграция с базой данных PostgreSQL
  • Обработка бронирований и отзывов
  • Уведомления через Telegram
  • Валидация данных
  • Защита от SQL-инъекций

Admin Panel (Strapi)

Функциональность

  • Управление меню - добавление, редактирование и удаление блюд и категорий
  • Управление мероприятиями - создание и редактирование событий
  • Модерация отзывов - одобрение или отклонение отзывов
  • Управление бронированиями - просмотр и обработка заявок
  • Настройка команды - информация о сотрудниках

Роли пользователей

  • Admin - полный доступ ко всем функциям
  • Content Manager - управление контентом
  • Editor - ограниченные права редактирования

База данных (PostgreSQL)

Структура базы данных

  • menu_categories - категории меню
  • menu_items - блюда меню
  • events - мероприятия
  • reviews - отзывы
  • reservations - бронирования
  • team_members - команда

Оптимизация

  • Индексы для быстрого поиска
  • Оптимизация запросов
  • Кэширование данных
  • Резервное копирование

Запуск проекта

Frontend (Next.js)

cd next-landing
npm install
npm run dev

Backend (Express.js)

cd backend
npm install
npm run dev

Admin Panel (Strapi)

cd admin
npm install
npm run develop
